## üìê Layout Principles

All components follow these design principles:

1. **Clean & Minimal**: Focus on content, minimal decoration
2. **Consistent Spacing**: Unified padding and margins
3. **Responsive**: Mobile-first design approach
4. **Accessible**: Proper contrast and interactive elements
5. **Performance**: Optimized rendering and lazy loading

## üé® Color Palette

- **Dark Text**: #111111 (main text color)
- **Gray Text**: #666666 (secondary text)

- **Border Gray**: #e5e7eb (dividers)

## üìè Typography

- **Headings**: System font stack with Roboto Slab for accent
- **Body Text**: Open Sans for readability
- **Sizes**: Responsive scaling based on viewport
- **Line Height**: Optimized for readability

## üöÄ Best Practices

1. Always use the existing components rather than creating new ones
2. Follow the established color palette
3. Maintain consistent spacing using Tailwind classes
4. Ensure all interactive elements have proper hover states
5. Test components across different screen sizes
6. Keep accessibility in mind (ARIA labels, keyboard navigation)
