# Migração para shadcn/UI

Este é o catálogo de componentes do projeto The Crypto Frontier que foram migrados para o design system baseado em shadcn/ui.

## Sobre a Migração

O projeto está passando por uma migração para utilizar o design system shadcn/ui, que oferece:

- Melhor suporte para temas claro/escuro
- Componentes acessíveis e bem testados
- Estilos consistentes e personalizáveis
- Melhor desempenho e manutenibilidade

## Componentes Migrados

Neste Storybook você encontrará demonstrações dos componentes já migrados, organizados nas seguintes categorias:

- **Componentes Básicos**: Button, Link, Badge, Avatar
- **Componentes de Formulário**: Input, Select, Checkbox, Form
- **Componentes de Layout**: Layout, PostLayout, PostFeedLayout
- **Blocos de Conteúdo**: ImageBlock, FormBlock, SearchBlock, TitleBlock, VideoBlock

Cada componente demonstra a versão original e a versão migrada, junto com as melhorias implementadas e guias de migração.
